## Prerequisites

- Minecraft Java Edition 1.21 or later
- A valid Minecraft account

## Server Address

Connect to: **play.titansmp.de**

## Step-by-Step

1. Open Minecraft and click **Multiplayer**.
2. Click **Add Server**.
3. Enter `play.titansmp.de` as the server address.
4. Click **Done** and then double-click the server to join.

## Troubleshooting

If you cannot connect, make sure:

- You are using Minecraft **Java Edition** (not Bedrock).
- Your client is updated to **1.21 or later**.
- You have a stable internet connection.
